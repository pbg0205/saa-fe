# 향후 개선 사항 (Future Improvements)

## 1. 기능 개선

### 1.1 사용자 계정 시스템
- 로그인/회원가입 기능
- 소셜 로그인 지원 (Google, GitHub 등)
- 사용자 프로필 관리

### 1.2 학습 진도 추적
- 문제별 풀이 여부 체크
- 학습 진행률 표시

### 1.3 오답 노트
- 틀린 문제 자동 저장
- 오답만 다시 풀기 기능
- 오답률 통계

### 1.4 문제별 해설 추가
- 상세한 문제 해설
- 왜 틀렸는지 설명
- 관련 AWS 서비스 문서 링크

### 1.5 카테고리별 문제 분류
- AWS 서비스별 분류 (EC2, S3, RDS 등)
- 주제별 분류 (네트워킹, 보안, 스토리지 등)
- 카테고리별 학습 진도 확인

### 1.6 현재 제출 답반 조회 기능
- 사용자가 시험에 제출한 답안 기록 저장
- 랜덤 문제 풀이 시 답안 확인 기능 활성화
- 모바일 기능 시 버튼 클릭시 제출한 답안 조회 모달 표시

### 1.7 북마크 기능
- 중요한 문제 북마크
- 북마크한 문제 모아보기
- 태그 기반 분류

## 2. UX 개선

### 2.1 다크 모드 지원
- 라이트/다크 모드 토글
- 시스템 설정 자동 감지
- 사용자 설정 저장

### 2.2 모바일 최적화
- 반응형 디자인 개선
- 터치 인터페이스 최적화
- 모바일 전용 네비게이션

### 2.3 키보드 단축키 지원
- 다음/이전 문제 이동 (이전 : [, 다음 :])
- 답안 선택 (숫자 키)
- 정답 확인 (Enter)
- 단축키 가이드 모달

### 2.4 문제 풀이 통계 대시보드
- 전체 정답률 표시
- 카테고리별 강점/약점 분석
- 최근 학습 활동 타임라인
- 학습 시간 통계

### 2.5 성취도 그래프
- 일일 학습량 그래프
- 정답률 추이 차트
- 카테고리별 레이더 차트
- 목표 달성률 시각화

## 3. 기술적 개선

### 3.1 데이터베이스 연동
- JSON 파일 → 데이터베이스 마이그레이션
- PostgreSQL 또는 MongoDB 사용
- Prisma ORM 도입
- 사용자 데이터 영구 저장

### 3.2 서버 상태 관리
- React Query 또는 SWR 도입
- 데이터 캐싱 전략
- Optimistic UI 업데이트
- 백그라운드 데이터 동기화

### 3.3 성능 최적화
- 이미지 최적화 (Next.js Image)
- 코드 스플리팅 개선
- 레이지 로딩 적용
- 번들 사이즈 최적화
- 메모이제이션 적용

### 3.4 접근성(A11y) 개선
- ARIA 레이블 추가
- 키보드 네비게이션 완전 지원
- 스크린 리더 호환성
- 색상 대비 개선
- 포커스 관리 개선

### 3.5 SEO 최적화
- 메타 태그 최적화
- Open Graph 태그 추가
- 사이트맵 생성
- 구조화된 데이터 (Schema.org)
- robots.txt 설정
